# Assistant-bot-RAG-
Был разработан Telegram-бот на Python с использованием Aiogram 3, в качестве AI-модели использовалось API OpenRouter.

Также был реализован аналогичный проект на базе n8n. Его ключевое преимущество — значительно более высокая скорость и простота разработки. При этом уровень функциональности и итоговые затраты у обоих решений сопоставимы.
